🎉 Welcome to the Intramurals Ranking System

This platform tracks medal counts for Manolo Fortich National High School Intramurals 2025 in real-time using Firebase.

🚀 Developed By

NanoNova Robotics Club – Programming Division

🎯 Purpose

To provide transparency, accessibility, and excitement by updating medal counts instantly.

💡 Features

  • Live medal tally
  • Smooth animations & ranking updates
  • Admin panel for updating scores
  • Mobile-friendly interface
